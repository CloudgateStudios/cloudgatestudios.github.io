Welcome to the newly rebuilt Cloudgate Studios website.

The old site had been running on a Bootstrap 3 theme since 2014 so it was overdue for a refresh. This new version is built with [Eleventy](https://www.11ty.dev/), a fast and flexible static site generator. All content is written in Markdown, which makes future updates a lot easier.

## What's new

- Fresh design with the updated Cloudgate Studios branding
- A blog (you're reading it)
- Cleaner, faster, and more maintainable code underneath

More posts coming soon — including updates on the apps and whatever else is being worked on.
